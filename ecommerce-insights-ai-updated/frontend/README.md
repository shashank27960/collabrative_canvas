
# Frontend (Vite + React) - Updated with Router and Improved UI

## Quick start
1. Ensure Node.js (v16+) and npm are installed.
2. From /frontend run:
   ```bash
   npm install
   npm run dev
   ```
3. Open the dev URL printed by Vite (usually http://localhost:5173) and ensure the backend is running on http://localhost:8000

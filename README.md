# Collaborative Canvas - Full Package (Upgraded)

This package contains a full upgraded implementation of the Real-Time Collaborative Drawing Canvas.

Folders:
- client/  -> frontend (HTML/CSS/JS)
- server/  -> Node.js + Socket.io + SQLite persistence
- deploy/  -> deployment configs & notes

Run:
1. cd server
2. npm install
3. node server.js
4. Open http://localhost:3000
